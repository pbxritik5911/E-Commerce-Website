# 🛍️ RAS E-Commerce Website

A modern, responsive, and feature-rich **E-commerce Website** built using **HTML, CSS, and JavaScript**.
This project simulates a real-world shopping experience with product browsing, search, cart management, and checkout flow.

---

## 🚀 Live Features

* 🛍️ Product Listing with Premium UI
* 🔍 Real-time Search Functionality
* 📄 Product Detail Page (Click to View)
* 🛒 Add to Cart System
* 🔢 Dynamic Cart Count & Price Calculation
* 💾 Cart Persistence using LocalStorage
* 🧾 Checkout Simulation
* 🎨 Clean Dark UI with Modern Design
